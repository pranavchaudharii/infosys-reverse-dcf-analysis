# 📊 Reverse DCF Analysis – Infosys

## 📌 Project Overview
This project performs a Reverse Discounted Cash Flow (DCF) analysis on Infosys to understand what growth expectations are currently priced into the stock.

## 🎯 Objective
To compare market-implied growth with historical financial performance and evaluate whether the stock is overvalued or fairly priced.

## 🔍 Key Insight
- Market-implied FCF growth: ~11%
- Historical Revenue Growth: ~9.2%
- Historical EPS Growth: ~5.2%

## 📈 Conclusion
The analysis suggests that Infosys is pricing in optimistic growth expectations, though not necessarily extreme overvaluation.

## 🧠 Key Concepts Used
- Discounted Cash Flow (DCF)
- Reverse DCF Analysis
- WACC (Weighted Average Cost of Capital)
- Terminal Value (Gordon Growth Model)
